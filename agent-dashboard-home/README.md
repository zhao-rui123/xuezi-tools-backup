## ✨ 功能特点

### 🦞 小龙虾角色
- 俯视像素形象
- 1-2秒走路动画移动到各房间
- 每个房间有不同的动作
- 情绪表情气泡
- 对话系统

### 🏠 智能房间切换
| 状态 | 房间 | 触发条件 |
|-----|------|---------|
| 睡觉中 | 🛏️ 卧室 | 深夜23:00-07:00 |
| 看电视 | 📺 客厅 | 休息中随机 |
| 游戏中 | 🎮 游戏室 | 休息10分钟后 |
| 工作中 | 💻 工作室 | 开始任务 |
| 吃饭中 | 🍽️ 餐厅 | 饭点/完成任务 |
| 摸鱼中 | 🚽 卫生间 | 随机触发15% |
| 运动中 | 🏃 运动场 | 高强度工作(>30分钟) |

### 📊 统计数据
- ✅ 今日完成任务数
- 🔥 Token消耗
- ⏱️ 工作时长
- 📍 当前位置

## 🚀 快速开始

### 访问看板
直接打开：http://106.54.25.161/agent-home/

### 在Python中记录任务
```python
import sys
sys.path.insert(0, '/Users/zhaoruicn/.openclaw/workspace/agent-dashboard-home')

# 需要集成tracker模块
def start_task(task_name):
    # 小龙虾移动到工作室
    print(f"🦞 开始任务: {task_name}")

def complete_task(tokens_used=0):
    # 小龙虾移动到客厅/餐厅/游戏室
    print(f"🦞 任务完成，消耗 {tokens_used} tokens")
```
