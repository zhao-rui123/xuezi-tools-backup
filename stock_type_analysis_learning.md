# 股票类型分析框架学习笔记

## 一、周期股分析方法

### 1.1 定义与特征
**周期股**：盈利能力随经济周期或行业周期大幅波动的股票
- 典型行业：锂矿、钢铁、煤炭、化工、航运、有色金属
- 核心特征：产品价格波动大 → 利润波动大 → 股价波动大

### 1.2 分析框架

#### 第一层：周期位置判断（最重要）
| 周期阶段 | 特征 | 投资策略 |
|---------|------|---------|
| **底部/复苏期** | 价格下跌→企稳、库存去化、产能收缩 | 逐步建仓 |
| **上升期** | 价格上涨、供不应求、产能扩张 | 持有 |
| **顶部/繁荣期** | 价格高位、产能大量释放、资本开支增加 | 减仓 |
| **下降期** | 价格下跌、库存累积、产能过剩 | 观望 |

#### 第二层：供需分析（周期核心）
**供给端：**
- 全球/国内产能利用率
- 新增产能投产时间表
- 产能退出情况（环保、亏损关停）
- 库存水平（产业链各环节）

**需求端：**
- 下游行业景气度（新能源车→锂矿）
- 替代品威胁
- 宏观经济关联度

#### 第三层：价格分析
- 现货价格走势
- 期货价格（远期预期）
- 成本曲线（支撑价格底线）

#### 第四层：估值方法（不同于成长股）
**周期股不能用PE估值！**

| 估值方法 | 适用阶段 | 说明 |
|---------|---------|------|
| **PB-ROE** | 全周期 | 核心指标，但要结合周期位置 |
| **EV/产能** | 底部/复苏期 | 重置成本法 |
| **分位估值** | 顶部判断 | 与历史高点对比 |
| **资源量估值** | 资源型企业 | 单位资源市值 |

### 1.3 锂矿股具体分析框架

```
锂矿股分析 = 碳酸锂周期 + 新能源车需求 + 公司成本优势
```

**关键跟踪指标：**
1. **碳酸锂价格**（电池级/工业级，日度跟踪）
2. **新能源车销量**（月度，特别是国内+欧洲）
3. **全球锂资源供给**（澳矿、南美盐湖、非洲矿山投产进度）
4. **库存数据**（冶炼厂、正极材料厂库存）
5. **成本曲线**（各家矿山现金成本，判断价格底部）

**周期判断示例：**
- 2020-2022年：新能源车爆发→碳酸锂涨价→锂矿股暴涨
- 2023-2024年：产能大量释放→碳酸锂跌价→锂矿股暴跌
- 2025-2026年？：取决于新能源车增速vs产能释放速度

## 二、成长股分析方法

### 2.1 定义与特征
**成长股**：处于高速成长期，盈利增速远高于平均水平的公司
- 典型行业：科技、新能源、生物医药、互联网
- 核心特征：高增速、高估值、再投资需求大

### 2.2 分析框架

#### 第一层：成长空间（天花板）
- **TAM（总可触达市场）**：市场空间多大？
- **渗透率**：当前处于什么阶段？（0-10%早期，10-30%成长期，30%+成熟期）
- **增速**：收入/利润 3年复合增速预期

#### 第二层：竞争优势（护城河）
- 技术壁垒（专利、研发投入）
- 客户粘性（转换成本）
- 网络效应
- 规模效应

#### 第三层：盈利质量
- 收入确认方式（是否可持续）
- 毛利率趋势（规模效应还是竞争加剧）
- 现金流（是否烧钱换增长）

#### 第四层：估值方法
| 指标 | 适用场景 | 说明 |
|------|---------|------|
| **PEG** | 首选 | PE/Growth，<1低估，1-2合理，>2高估 |
| **PS（市销率）** | 亏损期 | 收入增速 > 估值水平 |
| **远期PE** | 成熟期预期 | 预测成熟期利润，折现到现在 |
| **DCF** | 稳定成长期 | 现金流折现（参数敏感） |

### 2.3 成长股投资要点
- **早期（0-10%渗透）**：看空间，容忍高估值
- **成长期（10-30%渗透）**：看增速，PEG为主
- **成熟期（30%+渗透）**：看盈利，切换为价值股逻辑

## 三、价值股分析方法

### 3.1 定义与特征
**价值股**：业务成熟稳定、现金流好、估值偏低的公司
- 典型行业：银行、保险、公用事业、传统消费、地产
- 核心特征：低估值、高分红、低增长

### 3.2 分析框架

#### 第一层：估值安全边际
- **PE < 行业平均**（或历史PE下沿）
- **PB < 1**（或历史PB下沿）
- **股息率 > 无风险利率**（如3%+）

#### 第二层：资产质量
- **ROE稳定性**：连续5年ROE > 10%
- **负债率**：健康水平（行业对比）
- **现金流**：经营现金流/净利润 > 1

#### 第三层：分红能力
- **分红率**：30-70%合理区间
- **分红稳定性**：连续5年分红不减少
- **股息率**：绝对收益视角

#### 第四层：催化剂（可选）
- 行业政策变化
- 公司改革/转型
- 估值修复机会

### 3.3 价值股估值方法
| 指标 | 核心地位 | 标准 |
|------|---------|------|
| **PB-ROE** | 核心 | ROE > 15%，PB < 2 为优 |
| **股息率** | 核心 | > 4%有吸引力 |
| **PE** | 辅助 | < 15倍 |
| **自由现金流收益率** | 辅助 | FCF/市值 > 5% |

## 四、三类股票对比总结

| 维度 | 周期股 | 成长股 | 价值股 |
|------|--------|--------|--------|
| **核心分析** | 周期位置、供需 | 成长空间、增速 | 估值、资产质量 |
| **关键指标** | 产品价格、产能利用率 | 收入增速、渗透率 | ROE、PB、股息率 |
| **估值方法** | PB、EV/产能、分位 | PEG、PS、远期PE | PB-ROE、股息率 |
| **PE适用性** | ❌ 不适用 | ✅ 核心指标 | ⚠️ 辅助参考 |
| **投资时机** | 周期底部买入 | 早期/成长期 | 低估值时 |
| **持有周期** | 1-3年（周期一轮） | 3-5年（成长期） | 长期持有收息 |
| **风险** | 周期误判、价格暴跌 | 增速放缓、估值杀 | 价值陷阱、低增长 |

## 五、中矿资源（锂矿股）的正确分析框架

### 5.1 股票类型定位
**中矿资源 = 周期股（锂矿）**
- 不是成长股（虽然新能源车成长，但锂矿本身周期性强）
- 不是价值股（盈利波动大，分红不稳定）

### 5.2 应该这样分析

**第一步：周期位置判断**
- 碳酸锂价格处于什么阶段？（底部/反弹/高位/下跌）
- 全球锂资源供给释放进度？
- 新能源车需求增速是否支撑？

**第二步：供需分析**
- 供给：澳矿、南美盐湖、非洲矿山投产情况
- 需求：国内新能源车销量、库存去化情况
- 成本：公司现金成本vs行业成本曲线位置

**第三步：估值（周期股方法）**
- ❌ 不能用PE 137倍说高估
- ✅ 应该用：
  - PB 4.75倍 vs 历史PB分位
  - EV/产能 vs 重置成本
  - 资源量估值（每吨LCE市值）

**第四步：投资策略**
- 如果是周期底部：可以左侧布局，忍受短期亏损
- 如果是周期顶部：即使PE看起来低也要卖出
- 关键：判断碳酸锂价格方向，而不是看PE
