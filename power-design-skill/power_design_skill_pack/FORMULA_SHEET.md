
# 电力设计公式速查表

## 一、负荷计算

### 1. 需要系数法

**基本公式：**
```
Pc = Kd × Pe                          (有功功率, kW)
Qc = Pc × tanφ                        (无功功率, kvar)
Sc = √(Pc² + Qc²)                     (视在功率, kVA)
Ic = Sc / (√3 × Un)                   (计算电流, A)
```

**参数说明：**
- Pc: 计算有功功率 (kW)
- Qc: 计算无功功率 (kvar)
- Sc: 计算视在功率 (kVA)
- Ic: 计算电流 (A)
- Pe: 设备总容量 (kW)
- Kd: 需要系数
- cosφ: 功率因数
- tanφ: 功率因数角的正切值
- Un: 额定电压 (kV)

### 2. 利用系数法

**基本公式：**
```
Pav = Ku × Pe                         (平均有功功率, kW)
Pc = Km × Pav                         (计算有功功率, kW)
```

**参数说明：**
- Pav: 平均有功功率 (kW)
- Ku: 利用系数
- Km: 最大系数

### 3. 单位指标法

**基本公式：**
```
Pc = Pe' × A                          (有功功率, kW)
```

**参数说明：**
- Pe': 单位指标 (W/m²)
- A: 建筑面积 (m²)

## 二、短路电流计算

### 1. 标幺值法（高压系统）

**基准值：**
```
Ib = Sb / (√3 × Ub)                   (基准电流, kA)
Zb = Ub² / Sb                         (基准阻抗, Ω)
```

**各元件阻抗标幺值：**
```
Xs* = Sb / Ss                         (系统阻抗)
Xt* = (uk% / 100) × (Sb / Sn)        (变压器阻抗)
Xl* = Xl / Zb                         (线路阻抗)
```

**短路电流：**
```
Ik = Ib / Xtotal*                     (三相短路电流, kA)
Ik2 = (√3 / 2) × Ik                   (两相短路电流, kA)
Sk = √3 × Un × Ik                     (短路容量, MVA)
```

### 2. 有名值法/欧姆法（低压系统）

**基本公式：**
```
Ik = c × Un / (√3 × Zk)               (三相短路电流, kA)
```

**参数说明：**
- c: 电压系数 (低压取1.05)
- Un: 系统标称电压 (kV)
- Zk: 短路回路总阻抗 (mΩ)

**变压器阻抗：**
```
Zt = (uk% / 100) × (Un² / Sn) × 1000  (mΩ)
```

## 三、电缆截面选择

### 1. 按载流量选择

**校正系数：**
```
Kt = √((θn - θa) / (θn - θc))       (温度校正系数)
```

**参数说明：**
- θn: 电缆线芯长期允许工作温度 (℃)
- θa: 敷设处环境温度 (℃)
- θc: 基准温度 (℃)

### 2. 电压降计算

**基本公式：**
```
ΔU = √3 × I × L × (Rcosφ + Xsinφ)    (电压降, V)
ΔU% = (ΔU / Un) × 100%               (电压降百分比)
```

**参数说明：**
- I: 线路电流 (A)
- L: 线路长度 (km)
- R: 单位长度电阻 (Ω/km)
- X: 单位长度电抗 (Ω/km)
- cosφ: 功率因数
- sinφ: √(1 - cos²φ)

**电阻计算：**
```
R = ρ / S                             (Ω/km)
```

**参数说明：**
- ρ: 电阻率 (铜: 0.0172 Ω·mm²/m, 铝: 0.0283 Ω·mm²/m)
- S: 截面积 (mm²)

### 3. 热稳定校验

**基本公式：**
```
Smin = (I∞ × √t) / C                  (最小截面积, mm²)
```

**参数说明：**
- I∞: 短路电流 (A)
- t: 保护动作时间 (s)
- C: 热稳定系数 (铜: 143, 铝: 94)

## 四、继电保护整定

### 1. 电流速断保护

**基本公式：**
```
Iop = Krel × Ik.max                   (速断保护定值, A)
```

**参数说明：**
- Krel: 可靠系数 (1.2~1.5)
- Ik.max: 最大短路电流 (A)

### 2. 过电流保护

**基本公式：**
```
Iop = (Krel × Kast / Kr) × IL.max     (过流保护定值, A)
```

**参数说明：**
- Krel: 可靠系数 (1.15~1.25)
- Kast: 自启动系数 (1.5~2.5)
- Kr: 返回系数 (0.85~0.95)
- IL.max: 最大负荷电流 (A)

### 3. 灵敏度校验

**基本公式：**
```
Ksen = Ik.min / Iop                   (灵敏度系数)
```

**要求：**
- 速断保护: Ksen ≥ 2.0
- 过流保护: Ksen ≥ 1.5
- 远后备保护: Ksen ≥ 1.2

### 4. 时间级差

**基本公式：**
```
t1 = t2 + Δt                          (本级动作时间, s)
```

**参数说明：**
- t2: 下级保护动作时间 (s)
- Δt: 时间级差 (微机保护0.3s, 电磁继电器0.4s)

## 五、无功功率补偿

### 1. 补偿容量计算

**基本公式：**
```
Qc = P × (tanφ1 - tanφ2)              (补偿容量, kvar)
```

**参数说明：**
- P: 有功功率 (kW)
- tanφ1: 补偿前功率因数角的正切值
- tanφ2: 补偿后功率因数角的正切值

**正切值计算：**
```
tanφ = √(1 - cos²φ) / cosφ
```

### 2. 电容器实际输出

**基本公式：**
```
Q = QN × (U / UN)²                    (实际输出容量, kvar)
```

**参数说明：**
- QN: 额定容量 (kvar)
- U: 实际电压 (kV)
- UN: 额定电压 (kV)

### 3. 变压器补偿估算

**基本公式：**
```
Qc = S × β × k                        (补偿容量, kvar)
```

**参数说明：**
- S: 变压器容量 (kVA)
- β: 负载率
- k: 补偿率 (0.3~0.5)

## 六、接地设计

### 1. 水平接地极

**基本公式：**
```
R = (ρ / (2πL)) × ln(L² / (hd))      (接地电阻, Ω)
```

**参数说明：**
- ρ: 土壤电阻率 (Ω·m)
- L: 接地极长度 (m)
- h: 埋设深度 (m)
- d: 接地极直径 (m)

### 2. 垂直接地极

**基本公式：**
```
R = (ρ / (2πL)) × ln(4L / d)          (接地电阻, Ω)
```

### 3. 接地网

**基本公式：**
```
R = 0.5 × ρ / √S                      (接地电阻, Ω)
```

**参数说明：**
- S: 接地网面积 (m²)

## 七、变压器选择

### 1. 按负荷选择

**基本公式：**
```
Sn ≥ Pc / β                           (变压器容量, kVA)
```

**参数说明：**
- Pc: 计算负荷 (kVA)
- β: 负载率 (一般0.7~0.85)

### 2. 经济负载率

**基本公式：**
```
β = √(P0 / Pk)                        (经济负载率)
```

**参数说明：**
- P0: 空载损耗 (kW)
- Pk: 负载损耗 (kW)

### 3. 年电能损耗

**基本公式：**
```
ΔW = (P0 + β² × Pk) × T               (年电能损耗, kWh)
```

**参数说明：**
- T: 运行小时数 (h)

### 4. 电压调整率

**基本公式：**
```
ΔU% = β × (uk% × cosφ + ur% × sinφ)   (电压调整率, %)
```

**参数说明：**
- uk%: 阻抗电压百分比
- ur%: 电阻电压百分比

## 八、断路器选择

### 1. 额定电流

**选择原则：**
```
In ≥ 1.25 × Ic                        (额定电流, A)
```

**参数说明：**
- Ic: 计算电流 (A)

### 2. 分断能力

**选择原则：**
```
Icu ≥ 1.2 × Ik                        (分断能力, kA)
```

**参数说明：**
- Ik: 短路电流 (kA)

### 3. 电动机保护

**速断整定：**
```
Iop = (8~15) × In                     (笼型电动机)
Iop = (3~6) × In                      (绕线型电动机)
```

## 十、设计规范限值

### 电压降限值

| 回路类型 | 允许电压降 |
|---------|-----------|
| 照明回路 | ≤3% |
| 动力回路 | ≤5% |
| 特殊设备 | 按设备要求 |

### 接地电阻限值

| 系统类型 | 接地电阻 |
|---------|---------|
| 低压系统 | ≤4Ω |
| 高压系统(10kV) | ≤10Ω |
| 高压系统(35kV) | ≤4Ω |

### 变压器负载率

| 运行方式 | 负载率 |
|---------|--------|
| 经济运行 | 60%~70% |
| 正常负载 | 70%~85% |
| 短时过载 | ≤120% |

### 保护灵敏度

| 保护类型 | 灵敏度要求 |
|---------|-----------|
| 电流速断 | ≥2.0 |
| 过电流保护 | ≥1.5 |
| 远后备保护 | ≥1.2 |
