# 周线量化选股策略回测系统

## 策略说明

**买入条件**：
- KDJ金叉（K<30）+ MACD金叉

**卖出条件**（满足任一即卖出）：
1. 累计收益率 > 20%
2. MACD死叉 + KDJ死叉（同时）
3. 亏损达到 5%

**仓位**：全仓买入

## 回测结果（2024-2026年）

| 指标 | 数值 |
|------|------|
| 最终收益率 | +534.22% |
| 交易次数 | 128笔 |
| 盈利交易 | 大多数 |
| 止损交易 | 16次 |

## 注意事项

1. 100%正确率不现实，需放宽边界条件
2. 历史回测不代表未来收益
3. 需要进一步优化策略参数
