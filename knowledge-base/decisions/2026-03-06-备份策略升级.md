## 背景

原备份脚本 `backup_memory.sh` 仅备份：
- `memory/` - 对话记录
- `skills/` - 系统技能包

缺少：
- OpenClaw 配置文件
- 定时任务配置
- 完整的恢复文档

---

## 决策内容

### 1. 创建 system-backup 技能包

新建全面的备份解决方案，包含：

**每日增量备份** (`daily-backup.sh`)
- Memory 文件
- Skills（系统 + Workspace）
- OpenClaw 配置（openclaw.json、cron、auth-profiles）

**月度归档备份** (`monthly-archive.sh`)
- 完整 `~/.openclaw/` 目录 tar.gz 打包
- 自动保留最近 6 个月
- 适合灾难恢复和系统迁移

**恢复工具** (`restore.sh`)
- 从每日备份恢复
- 从月度归档恢复
- 仅恢复配置
- 完整的恢复文档

### 2. 定时任务更新

```cron
# 每日 22:00 - 增量备份
0 22 * * * skills/system-backup/scripts/daily-backup.sh

# 每月 1号 03:00 - 归档打包  
0 3 1 * * skills/system-backup/scripts/monthly-archive.sh

# 每日 22:05 - 备份检查
5 22 * * * scripts/backup-check.sh
```

### 3. 备份位置

```
/Volumes/cu/ocu/
├── memory/              # Memory 文件
├── skills/              # 系统技能包
├── workspace-skills/    # 自定义技能包
├── openclaw-config/     # OpenClaw 配置
└── archives/            # 月度归档 tar.gz
```
