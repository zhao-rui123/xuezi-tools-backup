## 可用模型

| 模型 | Provider | 特点 | 成本 |
|------|----------|------|------|
| MiniMax-M2.5 | minimax-cn | 主模型，reasoning | $0.3/M |
| kimi-k2.5 | bailian | 支持识图 | 免费 |
| qwen3.5-plus | bailian | 备用 | 免费 |
| kimi-for-coding | kimi-coding | Coding专用 | 免费 |

## 切换命令

```bash
# 切换到 kimi-k2.5（支持识图）
openclaw config set agents.defaults.model.primary "bailian/kimi-k2.5"
openclaw gateway restart

# 切换到 MiniMax
openclaw config set agents.defaults.model.primary "minimax-cn/MiniMax-M2.5"
openclaw gateway restart

# 切换到 qwen3.5-plus
openclaw config set agents.defaults.model.primary "bailian/qwen3.5-plus"
openclaw gateway restart

# 切换到 kimi-for-coding
openclaw config set agents.defaults.model.primary "kimi-coding/kimi-for-coding"
openclaw gateway restart
```

## 配置验证

```bash
# 查看当前配置
cat ~/.openclaw/openclaw.json | grep -A2 "primary"

# 查看 Gateway 状态
openclaw gateway status

# 查看模型列表
openclaw models list
```

## 故障排查

### 模型不允许错误
1. 检查 `openclaw.json` 是否包含该模型
2. 检查 `models.json` 是否配置了该 provider
3. 确认 API key 有效

### Gateway 连接失败
```bash
# 检查端口占用
lsof -i :18789

# 重启 Gateway
openclaw gateway restart

# 查看日志
openclaw logs
```

## 配置文件位置

| 文件 | 路径 |
|------|------|
| 主配置 | `~/.openclaw/openclaw.json` |
| 模型配置 | `~/.openclaw/agents/main/agent/models.json` |
| 认证配置 | `~/.openclaw/agents/main/agent/auth-profiles.json` |

## API Key 格式

- **MiniMax**: `sk-api-...` 或 `sk-cp-...`
- **百炼**: `sk-sp-...`
- **Kimi**: `sk-yIG...`
