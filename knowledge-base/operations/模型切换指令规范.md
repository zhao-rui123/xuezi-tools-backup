# 模型切换指令规范

**状态**: ✅已生效
**标签**: #OpenClaw #模型 #指令规范
**更新日期**: 2026-03-04

## 快捷指令映射

| 指令 | 模型 | Provider | 特点 |
|------|------|----------|------|
| `/model k2.5` | kimi-k2.5 | bailian | 百炼Kimi，支持识图 |
| `/model code` | kimi-for-coding | kimi-coding | 当前助手 |
| `/qwen` | qwen3.5-plus | bailian | 百炼千问 |
| `/minimax` | MiniMax-M2.5 | minimax-cn | 推理模型 |

## 文字切换指令

当用户使用自然语言切换模型时，按以下规则识别：

| 用户输入 | 切换到 | 说明 |
|----------|--------|------|
| "切换 kimi k2.5" / "切到 k2.5" | bailian/kimi-k2.5 | 百炼的Kimi |
| "切换 kimi code" / "切到 code" | kimi-coding/kimi-for-coding | 当前助手 |
| "切换千问" / "切到 qwen" | bailian/qwen3.5-plus | 百炼千问 |
| "切换 minimax" | minimax-cn/MiniMax-M2.5 | MiniMax |

## 禁止的模糊指令

以下指令**不明确**，需要向用户确认：

| 模糊指令 | 问题 | 正确做法 |
|----------|------|----------|
| "切换 kimi" | 不知道是 k2.5 还是 code | 询问用户具体要哪个 |
| "/model kimi" | 同上 | 提示使用 `/model k2.5` 或 `/model code` |

## 配置命令

```bash
# /model k2.5 -> 百炼 kimi-k2.5
openclaw config set agents.defaults.model.primary "bailian/kimi-k2.5"

# /model code -> kimi-for-coding (当前助手)
openclaw config set agents.defaults.model.primary "kimi-coding/kimi-for-coding"

# /qwen -> 百炼千问
openclaw config set agents.defaults.model.primary "bailian/qwen3.5-plus"

# /minimax -> MiniMax
openclaw config set agents.defaults.model.primary "minimax-cn/MiniMax-M2.5"
```

## 所有模型必须遵守

1. 收到 `/model k2.5` → 切换到 bailian/kimi-k2.5
2. 收到 `/model code` → 切换到 kimi-coding/kimi-for-coding
3. 收到模糊指令 "kimi" → 询问用户是 k2.5 还是 code
4. 切换后重启 Gateway: `openclaw gateway restart`
