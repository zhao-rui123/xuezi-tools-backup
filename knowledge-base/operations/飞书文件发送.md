## 核心规则

### ✅ 正确的文件路径

| 场景 | 正确路径 | 说明 |
|------|----------|------|
| 生成的文件 | `~/.openclaw/workspace/` | 截图、报告等 |
| 转发用户文件 | `~/.openclaw/media/inbound/` | 用户发给我的 |

### ❌ 错误的路径

| 路径 | 问题 |
|------|------|
| `/tmp/` | 飞书接收失败 |
| `/var/tmp/` | 权限问题 |
| 绝对路径 `/tmp/...` | 同样失败 |

## 截图发送流程

```bash
# 1. 截图保存到 workspace
/usr/sbin/screencapture -x ~/.openclaw/workspace/screenshot.png

# 2. 发送（通过 message 工具）
# media: /Users/zhaoruicn/.openclaw/workspace/screenshot.png
```

## 文档发送示例

```javascript
// 发送文本文件
{
  "action": "send",
  "caption": "文档名称",
  "media": "/Users/zhaoruicn/.openclaw/workspace/filename.txt"
}

// 转发用户发给我的图片
{
  "action": "send",
  "caption": "转发图片",
  "media": "/Users/zhaoruicn/.openclaw/media/inbound/[uuid].png"
}
```

## 支持的文件类型

- ✅ 图片：png, jpg, jpeg, gif
- ✅ 文档：txt, md, docx, xlsx, pdf
- ✅ 其他：zip, tar.gz
