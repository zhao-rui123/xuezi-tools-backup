# 电力系统短路计算 - 常见错误与正确方法

## 记录时间
2026-03-13

## 问题背景
110kV降压变电站短路计算及保护整定

## 犯的错误

### 错误1：系统阻抗计算错误
**错误做法：**
- 从35kV侧短路电流12kA反推系统阻抗

**正确做法：**
- 从110kV系统短路容量直接计算
- 公式：Xs = Sj / Ssc = 100 / 2000 = 0.05

### 错误2：变压器阻抗处理错误
**错误做法：**
- 直接用单台变压器阻抗Xt = 0.21

**正确做法：**
- 最大运行方式按两台变压器并联
- Xt_parallel = Xt / 2 = 0.21 / 2 = 0.105

### 错误3：过电流保护额定电流选择错误
**错误做法：**
- 用两台变压器总容量100MVA计算额定电流

**正确做法：**
- 用单台变压器容量50MVA计算
- Ie = 50 / (√3 × 35) = 824.8 A

### 错误4：保护整定公式遗漏
**错误做法：**
- 公式：Iop = Kk × Ie / (Kf × nTA)
- 漏了接线系数Kjx

**正确做法：**
- 完整公式：Iop = Kk × Kjx × Ie / (Kf × nTA)
- Kjx = 1.0（三相星形接线）

## 正确计算流程

### 短路电流计算步骤

```
步骤1：计算系统阻抗
   Xs = Sj / Ssc

步骤2：计算变压器阻抗（单台）
   Xt = (Uk%/100) × (Sj/Sn)

步骤3：考虑运行方式
   - 最大运行方式：两台并联，Xt_parallel = Xt / 2
   - 最小运行方式：单台运行，Xt_single = Xt

步骤4：计算总阻抗
   Xtotal = Xs + Xt_parallel

步骤5：计算短路电流
   Id = Ij / Xtotal
```

### 过电流保护整定步骤

```
步骤1：计算额定电流（单台变压器）
   Ie = Sn / (√3 × Ue)

步骤2：计算动作电流
   Iop = Kk × Kjx × Ie / (Kf × nTA)
   
   其中：
   - Kk = 1.2（可靠系数）
   - Kjx = 1.0（接线系数，三相星形）
   - Kf = 0.85（返回系数）
   - nTA = CT变比
```

## 关键要点总结

| 项目 | 注意事项 |
|------|---------|
| 系统阻抗 | 从高压侧系统短路容量直接计算 |
| 变压器阻抗 | 最大运行方式要并联，最小方式单台 |
| 额定电流 | 保护整定用单台变压器容量 |
| 接线系数 | 不要漏了Kjx |
| 基准值 | Sj = 100MVA，电压取额定电压 |

## 验算方法

1. 短路电流合理性检查
   - 10kV侧：通常在20-40kA范围
   - 35kV侧：通常在10-20kA范围

2. 保护整定合理性检查
   - 应该大于额定电流，小于短路电流
   - 通常额定电流的1.5-3倍

## 参考公式

### 标幺值法基本公式
- 基准电流：Ij = Sj / (√3 × Uj)
- 系统阻抗：Xs = Sj / Ssc
- 变压器阻抗：Xt = (Uk%/100) × (Sj/Sn)
- 短路电流：Id = Ij / Xtotal

### 保护整定公式
- 过电流：Iop = Kk × Kjx × Ie / (Kf × nTA)
- 速断：Iop = Kk × Kjx × Id_max / nTA
