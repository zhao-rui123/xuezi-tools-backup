## 核心技能：工商业储能项目测算

### 使用方法
```
计算 [省份] [年份]年[月份]月 工商业储能
- 容量：X kWh
- 设备成本：X 元/kWh
- 施工成本：X 元/kWh
- 效率：X%
- DOD：X%
- 运营期：X年
```

### 数据来源
- **电价数据**: 服务器 `/electricity/` 目录实时提取
- **循环次数**: 双峰值法计算结果（内置在电价程序中）

### 已验证案例
| 省份 | 时间 | 容量 | 结果 |
|------|------|------|------|
| 河南 | 2026-02 | 1000kWh | 15年ROI 407%，回收期3.7年 |
| 山东 | 2026-02 | 1000kWh | 15年ROI 548%，回收期2.7年 |
| 浙江 | 2026-02 | 1000kWh | 第三方投资，业主分成20% |

## 电价数据更新流程

### 更新频率
每月（用户主动通知）

### 更新步骤
1. 用户告知需要更新的省份
2. 修改 `/electricity/` 下的数据文件
3. 重新部署到云服务器
4. 更新 GitHub 备份

### 数据来源
各省发改委/电网公司官方文件

## 备份与恢复

### 备份文件
- **文件名**: `xuezi-tools-backup.tar.gz` (2.0MB)
- **服务器位置**: `/usr/share/nginx/html/`
- **GitHub**: 仓库自动同步

### 恢复命令
```bash
tar -xzvf xuezi-tools-backup.tar.gz -C /usr/share/nginx/html/
chmod -R 755 /usr/share/nginx/html/*
chmod 644 /usr/share/nginx/html/*/index.html
```

## 注意事项

1. **GitHub Pages 限制**: 电价查询工具使用 iframe 嵌入主站，如主站无法访问会显示错误
2. **其他工具独立**: 除电价查询外，其他5个工具在备用站可独立正常使用
3. **离线使用**: 可下载压缩包本地使用
