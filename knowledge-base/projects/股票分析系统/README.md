### 技术指标

#### 均线系统
- **MA5** - 5日移动平均线
- **MA10** - 10日移动平均线  
- **MA20** - 20日移动平均线

#### 趋势判断
- 📈 **多头排列** - 当前价 > MA5 > MA10 > MA20
- 📉 **空头排列** - 当前价 < MA5 < MA10 < MA20
- ↔️ **震荡整理** - 其他情况

#### RSI指标
- 🌡️ **RSI(14)** - 14日相对强弱指标
- 🔥 **超买** - RSI > 70
- ❄️ **超卖** - RSI < 30
- ➖ **正常** - 30 ≤ RSI ≤ 70

#### 偏离度
- 当前价与MA20的偏离百分比
- 📈 偏离 > +5% - 短期可能回调
- 📉 偏离 < -5% - 短期可能反弹

### 估值指标（雪球数据）

| 指标 | 说明 | 判断标准 |
|------|------|---------|
| PE(TTM) | 市盈率 | <15低估, 15-30合理, 30-50偏高, >50高估 |
| PB | 市净率 | 越低越好 |
| 换手率 | 成交活跃度 | >3%活跃, <1%低迷 |
| 量比 | 相对成交量 | >1.5放量, <0.8缩量 |
| 52周位置 | 当前价格在52周区间的百分比 | 0-100% |

## 配置文件

### 雪球Cookie配置
位置：`stock_feishu_push_fusion.py` 头部

```python
XUEQIU_COOKIES = {
    'xq_a_token': '你的token',
    'xq_id_token': '你的id_token',
    # ...
}
```

**获取方法**:
1. 登录 [xueqiu.com](https://xueqiu.com)
2. F12打开开发者工具 → Network
3. 刷新页面，找到任意quote.json请求
4. 右键 → Copy → Copy as cURL (bash)
5. 提取其中的Cookie部分

### 自选股配置
位置：`stock_feishu_push_fusion.py` 中的 `WATCHLIST`

```python
WATCHLIST = [
    ("002738", "中矿资源", "锂矿/资源"),      # A股
    ("00981", "中芯国际", "半导体/芯片"),      # 港股
    # ...
]
```

**港股代码规则**: 5位数字，以0/1/2/3/6/8开头
