## 数据来源

### 新浪财经API
- **URL格式**: `https://hq.sinajs.cn/list=sh[代码],sz[代码]`
- **优点**: 免费、实时、无需认证
- **限制**: 有频率限制，需控制请求间隔

### 数据字段
| 字段 | 说明 |
|------|------|
| 名称 | 股票名称 |
| 最新价 | 当前价格 |
| 涨跌幅 | 当日涨跌百分比 |
| 成交量 | 当日成交量 |
| 成交额 | 当日成交金额 |
