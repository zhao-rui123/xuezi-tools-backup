# 小龙虾之家 - AI助手工作状态看板

| **工作室** | 💻 | 写代码、工作 |
| **茶水室** | 🍵 | 喝茶、摸鱼 |

| **游戏室** | 🎮 | 游戏、RGB氛围 |
| **餐厅** | 🍽️ | 吃饭 |
| **卫生间** | 🚽 | 摸鱼、思考 |
| **浴室** | 🚿 | 洗澡 |
| **主卧** | 🛏️ | 睡觉 |
| **阳台运动场** | 🏃 | 运动健身 |

### 2. 小龙虾角色 🦞

- **8-bit像素风格**设计
- **实时跟随**我的工作状态切换房间
- **动画效果**: 走路、发光、表情气泡

**房间切换规则**:
| 我的状态 | 小龙虾位置 | 触发 |
|----------|-----------|------|
| 写代码/开发 | 💻 工作室 | 工作时 |
| 休息/等待 | 🛋️ 客厅 | 空闲时 |
| 深夜工作 | 🛏️ 卧室 | 23点后 |
| 长时间思考 | 🍵 茶水室 | 思考中 |

## 技术实现

### 技术栈
| 层级 | 技术 |
|------|------|
| 前端 | HTML5 + CSS3 (像素风) + JavaScript |
| 后端 | Python Flask API |
| 数据 | JSON文件存储 |
| 部署 | 腾讯云服务器 + Nginx |
| 定时任务 | Linux Cron |

### 文件位置
```
/usr/share/nginx/html/my-home/    # 前端文件
/opt/lobster-home/                # API和数据
/opt/lobster-home/data/           # 统计数据
/opt/lobster-home/data/history/   # 历史归档
```

### 定时任务
```bash
# 每日00:00 - 数据归档
crontab: 0 0 * * * /usr/bin/python3 /opt/lobster-home/archive_daily.py

# 每日22:00 - 飞书推送
crontab: 0 22 * * * /usr/bin/python3 /opt/lobster-home/feishu_daily.py
```

## 数据持久化

### 自动归档流程
1. 每天00:00自动执行`archive_daily.py`
2. 将当天数据保存到`history/YYYY-MM-DD.json`
3. 重置今日数据为新的一天
4. 清理30天前的旧数据

### 数据格式
```json
{
  "date": "2026-03-08",
  "tasks_completed": 3,
  "tokens_used": 11000,
  "work_time_seconds": 7200,
  "tasks": [...],
  "room_time": {...}
}
```

- "虽然任务不多，但思考很深入~"
