# 电价查询系统 V4 - 项目文档

## 项目目标
1:1 复制现有电价查询网站，优化为易维护版本，添加2026年3月电价数据

## 现有网站分析
- **URL**: http://106.54.25.161/electricity/
- **技术栈**: 纯 HTML + CSS + JavaScript (无框架)
- **文件结构**:
  - index.html - 主页面
  - app.js - 主逻辑
  - price-data.js - 电价数据（31省份）
  - price-service.js - 数据服务
  - styles.css - 样式
  - api/price-data.js - API接口

## 核心功能
1. 省份选择（31个省份）
2. 年份/月份选择
3. 时段详情展示（尖峰/高峰/平段/低谷/深谷）
4. 储能循环次数计算
5. 电费数据表格
6. 图表展示

## 优化目标
1. **数据与代码分离** - 电价数据单独存放，便于更新
2. **模块化结构** - 清晰的功能模块划分
3. **易维护** - 只修改数据文件即可更新电价
4. **保留所有功能** - 1:1复制，不删减任何功能

## 数据更新规则
- 只修改 `price-data.js` 添加新月份数据
- 不修改 HTML/JS 代码
- 支持多个月份并存（1月、2月、3月...）
