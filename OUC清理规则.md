# OUC 文件夹清理规则

## 📋 清理策略总览

| 目录/文件类型 | 保留期限 | 清理周期 | 说明 |
|--------------|---------|---------|------|
| **skills-backup/** 技能包备份 | 30天 | 每周 | 自动备份文件，保留最近30个版本 |
| **archived/** 归档文件 | 90天 | 每月 | 旧压缩包、历史文件 |
| **generated/** 生成的文件 | 180天 | 每月 | AI生成的文档、方案（重要文件手动归档）|
| **downloads/** 下载文件 | 30天 | 每周 | 临时下载，及时清理 |
| **extracted/** 解压文件 | 7天 | 每天 | 临时解压，用完即删 |
| **memory/** 记忆文件 | 365天 | 每月 | 每日工作记录，保留1年 |
| **backups/** 其他备份 | 60天 | 每月 | 临时备份文件 |
| **.DS_Store** 缓存文件 | 立即 | 每次 | macOS系统缓存 |

---

## 🗂️ 详细清理规则

### 1. 技能包备份 (skills-backup/)

**保留策略：**
- 保留最近30天的备份
- 最多保留30个版本的tar.gz文件
- 超过自动删除

**为什么30天？**
- 技能包每天都有备份，30天足够回溯问题
- tar.gz文件较大，需要控制磁盘空间
- 重要版本可以手动复制到archived/长期保存

---

### 2. 归档文件 (archived/)

**保留策略：**
- 保留最近90天
- 旧的压缩包、历史版本
- 手动确认后删除

**注意：**
- 重要历史版本建议永久保留
- 删除前确认无依赖

---

### 3. 生成的文件 (generated/)

**保留策略：**
- 默认保留180天（6个月）
- 重要文档手动移动到documents/长期保存
- 临时方案、测试文件及时删除

**分类管理：**
```
generated/
├── 临时测试/          # 保留7天，用完即删
├── 方案草稿/          # 保留30天，确认后归档
└── 正式文档/          # 保留180天，重要文档转存documents/
```

---

### 4. 下载文件 (downloads/)

**保留策略：**
- 保留最近30天
- 下载后及时使用，用完即删
- 重要文件移动到对应目录

**建议：**
- 下载后立即解压/使用
- 不要长期堆积在downloads/

---

### 5. 解压文件 (extracted/)

**保留策略：**
- 保留最近7天
- 临时解压目录，用完即删
- 空目录自动删除

**为什么7天？**
- 解压后通常立即使用
- 7天足够完成操作
- 防止临时文件堆积

---

### 6. 记忆文件 (memory/)

**保留策略：**
- 保留最近365天（1年）
- 按日期命名的markdown文件
- 每年底归档到memory/archive/

**长期保存：**
- 重要决策转存到MEMORY.md
- 年底打包归档

---

### 7. 其他备份 (backups/)

**保留策略：**
- 保留最近60天
- 临时备份、手动备份
- 确认无用后删除

---

### 8. 系统缓存 (.DS_Store)

**保留策略：**
- 立即删除
- macOS系统自动生成
- 每次清理时删除

---

## ⏰ 清理执行计划

### 自动清理（推荐）

**设置每周日凌晨3点执行：**
```bash
# 添加到crontab
0 3 * * 0 /Users/zhaoruicn/.openclaw/workspace/scripts/ouc-cleanup.sh
```

**清理顺序：**
1. 清理缓存文件（.DS_Store）
2. 清理临时文件（extracted/）
3. 清理下载文件（downloads/）
4. 清理旧备份（skills-backup/）
5. 清理归档文件（archived/）
6. 清理记忆文件（memory/）
7. 清理生成的文件（generated/）
8. 删除空目录

### 手动清理

**每月检查一次：**
```bash
# 执行清理脚本
bash /Users/zhaoruicn/.openclaw/workspace/scripts/ouc-cleanup.sh

# 查看日志
cat /tmp/ouc-cleanup.log
```

## 📊 磁盘空间监控

### 预警阈值

| 磁盘使用率 | 状态 | 建议 |
|-----------|------|------|
| < 70% | 🟢 正常 | 按计划清理 |
| 70% - 85% | 🟡 警告 | 增加清理频率 |
| > 85% | 🔴 危险 | 立即清理，删除大文件 |

### 查看磁盘空间

```bash
# 查看OUC文件夹总大小
du -sh /Volumes/cu/ocu/

# 查看各目录大小
du -sh /Volumes/cu/ocu/*/ | sort -h

# 查看大文件
find /Volumes/cu/ocu -type f -size +100M
```

## ⚠️ 重要提示

### 删除前确认

1. **不要删除正在使用的文件**
2. **重要文件先备份**
3. **不确定的文件先移到archived/**

### 白名单（永不删除）

```
/Volumes/cu/ocu/
├── knowledge-base/     # 知识库（手动管理）
├── skills/            # 技能包（手动管理）
├── memory/            # 最近30天的记忆
├── documents/         # 重要文档（手动管理）
└── README.md          # 目录说明
```

### 紧急清理

**当磁盘空间不足时：**
```bash
# 1. 立即删除大文件
find /Volumes/cu/ocu -type f -size +500M -delete

# 2. 清空downloads/
rm -rf /Volumes/cu/ocu/downloads/*

# 3. 清空extracted/
rm -rf /Volumes/cu/ocu/extracted/*

# 4. 只保留最近7天的备份
find /Volumes/cu/ocu/skills-backup -mtime +7 -delete
```
