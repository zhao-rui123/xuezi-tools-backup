# Cursor AI 编程工具 - Win11 安装指南

## 什么是 Cursor？

Cursor 是一个 AI 编程工具，基于 VS Code 开发。

**特点：**
- 不用写代码，用自然语言描述需求，AI 帮你写
- 中文对话，0 基础也能用
- 免费额度够用
- 界面和 VS Code 类似，但更简单

---

## 安装步骤

### 第一步：下载 Cursor

1. 打开浏览器，访问：https://cursor.com
2. 点击 "Download for Windows"
3. 下载安装包（约 200MB）

### 第二步：安装

1. 双击下载的 `.exe` 文件
2. 按照提示点击 "下一步"
3. 建议勾选 "添加到右键菜单"（方便快速打开文件夹）
4. 点击 "安装"
5. 等待安装完成

### 第三步：初始设置

1. 打开 Cursor
2. 登录（可以用 GitHub 账号或邮箱注册）
3. 选择主题（建议选深色，对眼睛好）
4. 完成！

## 基本使用（0 基础入门）

### 1. 创建第一个项目

1. 点击 "Open Folder"（打开文件夹）
2. 在电脑上创建一个新文件夹，比如 `D:\我的项目\第一个网页`
3. 选择这个文件夹，点击 "选择文件夹"

### 2. 让 AI 帮你写代码

**按 `Ctrl + L` 打开 AI 对话框**

然后输入你的需求，例如：

```
帮我创建一个简单的个人介绍网页，
要有：
- 我的名字：张三
- 头像（用占位图）
- 个人简介
- 联系方式
要求：美观、响应式
```

Cursor 会自动生成代码，你只需要点击 "Accept"（接受）即可。

### 3. 预览效果

1. 按 `Ctrl + Shift + V` 预览网页
2. 或者右键点击 HTML 文件 → "Open with Live Server"

### 4. 修改代码

不满意的地方直接告诉 AI：

```
把背景颜色改成深蓝色
字体放大一点
```

AI 会自动修改代码。

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + L` | 打开 AI 对话框 |
| `Ctrl + K` | 快速编辑选中的代码 |
| `Ctrl + /` | 注释/取消注释 |
| `Ctrl + S` | 保存文件 |
| `Ctrl + Shift + V` | 预览网页 |

## 学习建议

### 第一阶段（第 1-3 天）：熟悉界面
- 创建几个简单网页
- 尝试修改颜色、文字、布局
- 不需要理解代码，先感受 AI 的能力

### 第二阶段（第 4-7 天）：尝试复杂项目
- 做个待办清单网页
- 做个计算器
- 做个个人博客首页

### 第三阶段（第 2 周起）：学习基础概念
- 了解 HTML 是结构
- 了解 CSS 是样式
- 了解 JavaScript 是交互
- 不需要精通，能看懂 AI 生成的代码即可

## 常见问题

### Q: 需要会英语吗？
**A:** 不需要。Cursor 支持中文对话，AI 能理解中文需求。

### Q: 免费版够用吗？
**A:** 够用。免费版每月 2000 次 AI 对话，初学者完全够。

### Q: 会替代程序员吗？
**A:** 不会。它是工具，让编程更简单，不是替代人。

### Q: 0 基础真的能学会吗？
**A:** 能。很多非程序员用 Cursor 做出了自己的网站、小程序。
