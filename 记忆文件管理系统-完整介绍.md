# 雪子助手记忆文件管理系统 - 完整介绍

## 系统概述

**记忆文件管理系统**是一个三层架构的智能记忆体系，用于持久化保存AI助手（我）与用户（雪子）的协作历史、工作记录、重要决策和技术方案。系统通过自动化工具实现记忆采集、智能分析、永久存档和快速检索，大幅减少未来查询时的Token消耗。

## 二、系统组件详解

### 2.1 日常记忆层

**位置**：`~/.openclaw/workspace/memory/`

**文件命名**：`YYYY-MM-DD.md`

**内容格式**：
```markdown
# YYYY-MM-DD 工作记录

## 完成的工作
1. **任务名称**
   - 详细描述
   - 关键决策
   - 遇到的问题和解决方案

## 关键数据
- Token消耗：X
- 工作时长：X小时
- 解锁成就：XXX

## 重要决策
- [DECISION] 决策内容

## TODO
- [ ] 待办事项
```

**特点**：
- 每天工作结束时保存
- 永久保留，不删除
- 详细记录当天所有工作
- 是系统的"原始数据"

---

### 2.2 智能分析层

**核心组件**：记忆智能筛选系统

**位置**：`~/.openclaw/workspace/skills/memory-processor/`

**主要功能**：

#### 2.2.1 关键词提取
- 使用TF-IDF算法
- 提取中文词汇（2-6字）
- 提取英文单词（3字符以上）
- 过滤停用词
- 输出TOP30关键词及频次

#### 2.2.2 主题识别
- 基于关键词聚类分析
- 预定义主题映射：
  - 储能项目
  - 股票分析
  - 零碳园区
  - 技能包开发
  - 记忆管理
  - 系统运维
  - 飞书集成
  - 项目测算
  - 小龙虾之家
  - 战略规划

#### 2.2.3 月度摘要生成
**输出位置**：`memory/summary/YYYY-MM-summary.md`

**内容**：
- 统计信息（文件数、字符数）
- 高频关键词TOP15（可视化条形图）
- 主题分布
- 永久记忆建议

#### 2.2.4 永久记忆存档
**输出位置**：`memory/permanent/YYYY-MM-permanent.md`

**内容**：
- 本月核心主题（TOP3）
- 相关关键词
- 快速检索链接

#### 2.2.5 索引更新
**关键词索引**：`memory/index/keywords.json`
```json
{
  "2026-03": {
    "date": "2026-03-08T21:32:47",
    "keywords": [
      ["api", 20],
      ["openclaw", 20],
      ["agent", 15],
      ...
    ]
  }
}
```

**主题索引**：`memory/index/themes.json`
```json
{

    "themes": {
      "技能包开发": 1,
```

## 十二、相关技能包

### 12.1 记忆智能筛选 (memory-processor)
- 每月自动分析记忆文件
- 提取高频关键词和主题
- 生成月度摘要和永久记忆
- 同步到知识库

### 12.2 会话持久化 (session-persistence)
- 切换模型时保存会话状态
- 恢复工作上下文
- 避免重复了解任务背景

### 12.3 增强记忆系统 (enhanced-memory) ⭐ 重要性分级
- **混合检索**：向量相似度 + BM25 文本搜索
- **时间衰减**：老记忆自动降级
- **重要性加权**：重要记忆优先（0-1评分）
- **噪声过滤**：自动过滤低质量内容
- **会话记忆**：`/new` 时自动提取保存关键信息

**重要性分级算法**：
```
最终得分 = 相似度得分 × 时间衰减 × 重要性加权

时间衰减 = 0.5 + 0.5 × exp(-年龄天数 / 60)
重要性加权 = 0.7 + 0.3 × importance
```

**使用方式**：
```python
# 存储记忆（带重要性评分）
store_memory(
    text="用户喜欢Python编程",
    category="preference",
    importance=0.8,    # 重要性 0-1
    scope="global"
)

# 检索记忆（自动按重要性排序）
recall_memory(query="用户编程偏好", top_k=5)
```
