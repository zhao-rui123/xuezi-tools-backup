## 2026-02-21 重要技能创建

### 工商业储能项目测算技能

**技能位置：** `/root/.openclaw/workspace/skills/storage-calc/`

**核心文件：**
- `SKILL.md` - 使用说明
- `calculator.js` - 核心计算逻辑

**使用方法：**
```
计算 [省份] [年份]年[月份]月 工商业储能
- 容量：X kWh
- 设备成本：X 元/kWh
- 施工成本：X 元/kWh
- 效率：X%
- DOD：X%
- 运营期：X年
```

**数据来源：**
- 电价数据：从电价查询程序实时提取（`/electricity/`）
- 循环次数：双峰值法计算结果

**计算逻辑：**
1. 从服务器提取指定省份、年份、月份的电价数据
2. 获取该省份年循环次数及循环类型分布
3. 计算有效放电量 = 容量 × DOD × 效率
4. 按循环类型分别计算套利收益
5. 输出财务指标（投资回收期、ROI等）
