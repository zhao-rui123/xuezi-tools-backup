#### [BREAKTHROUGH] 图生图功能
**发现**：MiniMax image-01支持`subject_reference`参数，可实现图生图！

**API地址**：`https://api.minimaxi.com/v1/image_generation`

**调用示例**：
```bash
curl -X POST "https://api.minimaxi.com/v1/image_generation" \
  -H "Authorization: Bearer <TokenPlan Key>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "image-01",
    "prompt": "描述新场景",
    "aspect_ratio": "1:1",
    "subject_reference": [
      {
        "type": "character",
        "image_file": "data:image/jpeg;base64,<图片base64>"
      }
    ]
  }'
```

**关键参数**：
- `model`: image-01 或 image-01-live
- `subject_reference[].type`: "character"
- `subject_reference[].image_file`: base64或公网URL
- `aspect_ratio`: 1:1, 16:9, 4:3等
- `style_type`: 漫画、元气、中世纪、水彩（仅live模型）
