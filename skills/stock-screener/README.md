# 股票月线趋势筛选工具

## 简介
基于月线级别的MACD+KDJ双金叉+均线多头排列+成交量放大的A股筛选工具，支持周期股预期差分析和卖出信号监控。

## 投资方法论

### 买入策略（三级筛选）

**A级（最佳）** ★
- 条件：MACD金叉 + KDJ金叉 + 均线多头排列 + 成交量放大
- 仓位建议：首次建仓不超过30%，确认趋势后分批加仓

**B级（可接受）** ☆
- 条件：MACD金叉 + 均线多头排列 + 成交量放大（KDJ滞后）
- 仓位建议：首次建仓不超过20%，等待KDJ金叉确认后加仓

**C级（观察）** ○
- 条件：KDJ金叉 + 均线多头排列 + 成交量放大（MACD滞后）
- 仓位建议：首次建仓不超过15%，等待MACD金叉确认后加仓

### 卖出/减仓策略

| 信号 | 条件 | 操作 |
|------|------|------|
| 偏离减仓 | 股价偏离5月均线 > 20% | 减仓1/3-1/2 |
| 偏离减仓 | 股价偏离5月均线 10-20% | 减仓1/4-1/3 |
| KDJ卖出 | KDJ > 80 且 死叉 | 减仓1/3-1/2 |
| 趋势破坏 | 月线MACD死叉 | 清仓或大幅减仓 |
| 止损 | 跌破20月均线 | 触发止损 |

### 回补策略
- 股价回踩5月均线附近（偏离度10-20%）分批回补
- 留足底仓，卖飞不追

### 周期股预期差
- **PE历史低位**：PE分位点 < 30%
- **深度调整**：股价较高点跌幅 > 50%
- **盈利预期差**：市场预期利润下滑，实际维持高位

### 风险控制
- 单一股票仓位 ≤ 30%
- 总仓位：牛市80% / 震荡市60% / 熊市40%

## 技术指标参数
- **MACD**: 12, 26, 9
- **KDJ**: 23, 3, 3
- **均线**: 5月、10月、20月
- **成交量**: 当月 > 前3月平均 × 1.2


## 运行筛选

### 基础筛选
```bash
python screener.py
```

### 完整分析（含基本面、卖出信号、周期股分析）
```bash
python full_analysis.py
```

## 配置文件
修改 `config.py` 可调整：
- 技术指标参数
- 成交量放大倍数
- KDJ买入区域上限
- 输出格式

## 输出结果
- JSON格式：完整数据包含所有指标
- CSV格式：表格形式便于查看
- TXT报告：可读性强的分析报告
- 文件位置：`output/screening_YYYYMMDD_HHMMSS.*`
