## 🎯 系统概述

**统一记忆管理系统 (Unified Memory System)** 是一个全功能的记忆管理技能包，整合了原先分散的多个记忆相关工具，提供一致的使用接口和智能化的记忆管理体验。

### 核心理念
- **一个命令**搞定所有记忆操作
- **自动识别**重要信息并存储
- **智能检索**优先显示重要内容
- **永久保存**工作历史不丢失

### 2. 双模式记忆存储

#### 模式一：自动识别存储
在对话中自动识别并存储重要信息：

| 识别模式 | 示例 | 存储类别 | 默认重要度 |
|---------|------|---------|-----------|
| "我喜欢..." | "我喜欢用 Python 编程" | preference | 0.7 |
| "我决定..." | "我决定使用 React" | decision | 0.9 |
| "我叫..." | "我叫雪子，做储能项目" | identity | 0.85 |
| "我的股票..." | "我的股票代码是 002460" | finance | 0.85 |
| "千万不要..." | "千万不要上传 API key" | constraint | 0.9 |
| "我计划..." | "我计划明天开会" | schedule | 0.75 |

#### 模式二：主动指令存储
```
用户：记住我的邮箱是 xxx@qq.com
→ 立即存储，重要度 0.85

命令行：ums recall store "内容" --importance 0.9
→ 精确控制存储参数
```

### 3. 智能检索系统

**搜索命令：**
```bash
ums recall search "关键词" --top 5
```

**检索特点：**
1. **混合匹配** - 支持关键词和类别搜索
2. **重要性优先** - 高分记忆排在前面
3. **时间衰减** - 老记忆自动降级
4. **类别过滤** - 可按类别筛选

**评分算法：**
```
最终得分 = 匹配度 × 时间衰减 × 重要性加权

时间衰减 = 0.5 + 0.5 × exp(-年龄天数 / 60)
重要性加权 = 0.7 + 0.3 × 重要度(0-1)
```

### 4. 会话持久化

**切换模型不丢失记忆：**
```bash
# 切换前保存
ums session save

# 切换模型后恢复
ums session restore
```

**保存内容：**
- 当前工作状态
- 最近记忆文件
- 系统配置信息

### 5. 月度智能分析

**每月1日 02:00 自动执行：**
```bash
ums analyze monthly
```

**分析内容：**
- 统计月度工作数据
- 提取 TOP30 高频关键词
- 识别核心工作主题
- 生成月度摘要报告
- 更新关键词/主题索引
- 同步高频内容到知识库

---

### 6. 预测性工作建议 (NEW! v2.1)

基于历史工作模式，主动预测未来工作重点，优化工作节奏。

**功能模块：**
- 📊 **下周重点预测** - 基于历史数据分析，预测下周应关注的工作主题
- ⏱️ **工作节奏建议** - 分析工作强度模式，提供节奏优化建议
- ⏰ **逾期事项提醒** - 扫描待办任务，识别可能已逾期的项目
- 📈 **工作模式分析** - 深入分析历史数据，识别趋势和洞察

**使用方式：**
```bash
# 生成每日简报（推荐）
ums advisor --action briefing

# 预测下周工作重点
ums advisor --action focus

# 工作节奏建议
ums advisor --action rhythm

# 逾期事项提醒
ums advisor --action remind

# 工作模式深度分析
ums advisor --action patterns
```
