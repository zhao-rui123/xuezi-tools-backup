## 应用场景

### 场景1: 零碳园区方案设计
```
踏勘 → 行业分析 → 能源审计 → 方案设计 → 投资估算 → 报告生成
```

### 场景2: 节能诊断
```
数据收集 → 能耗分析 → 节能潜力 → 措施建议 → 投资回收 → 实施计划
```

### 场景3: 光伏系统设计
```
选址 → 倾角计算 → 容量设计 → 组件选型 → 发电量估算 → 收益分析
```

### 场景4: 余热回收设计
```
热源识别 → 资源评估 → 技术选型 → 系统设计 → 投资估算 → 回收期
```

### 场景5: 谐波治理
```
谐波测试 → THD分析 → 影响评估 → 治理方案 → 设备选型 → 效果预测
```

## 技术特点

1. **专业性强**: 基于国家标准和行业规范
2. **覆盖面广**: 涵盖水电气暖、光伏风电、余热回收、建筑节能、谐波治理、碳排放
3. **行业深度**: 14个高耗能行业详细参数
4. **计算准确**: 采用标准公式和实测数据
5. **易于使用**: 纯Python实现，无额外依赖
6. **扩展性好**: 模块化设计，易于扩展

---

## 使用建议

1. **踏勘前**: 使用行业数据库了解目标行业特点
2. **踏勘中**: 参照踏勘指导收集关键数据
3. **踏勘后**: 使用各计算模块进行详细分析
4. **方案设计**: 使用SchemeDesigner生成综合方案
5. **报告编写**: 使用export_scheme_report生成报告
