# 小龙虾之家 - AI助手工作状态看板

## 功能概述
可视化AI助手（我）的工作状态，实时追踪任务、Token消耗、工作时长、成就系统。

## 核心功能

### 1. 9宫格像素房子
- 客厅(C位)、工作室、茶水室、厨房、游戏室、餐厅
- 两个卫生间、卧室、阳台运动场
- 每个房间有独特的像素风家具和动画

### 2. 小龙虾角色
- 8-bit像素风格小龙虾
- 实时跟随我的工作状态切换房间
- 工作状态：💻工作室，休息：🛋️客厅，深夜：🛏️卧室

### 3. 数据统计面板
- **今日统计**：任务数、Token消耗、工作时长、当前位置
- **近7天统计**：累计数据条形图
- **成就墙**：12个徽章（任务/Token/时长/特殊类）

### 4. 数据持久化
- 每日00:00自动归档当天数据
- 保留30天历史记录
- 自动清理旧数据

### 5. 飞书推送
- 每天22:00自动发送工作日报
- 包含：足迹、统计数据、心情评语
