# Stock Earnings Calendar - 财报日历

## 功能概述

追踪自选股财报披露日期，提前提醒重要事件。

## 功能特性

### 📅 财报追踪
- 年报/季报披露日期
- 业绩预警提醒
- 股东大会日期

### 📢 公告监控
- 增减持公告
- 定增公告
- 股权激励
- 投资/并购公告

### ⏰ 提醒设置
- 提前N天提醒（默认7天）
- 紧急事件即时提醒

## 使用方法

### 查看财报日历

```bash
cd ~/.openclaw/workspace/skills/stock-earnings-calendar
python3 earnings_calendar.py
```

### 检查近期事件

```python
from earnings_calendar import check_upcoming_earnings, format_earnings_calendar

# 检查未来7天的事件
events = check_upcoming_earnings(days_ahead=7)
print(format_earnings_calendar(events))
```

## 配置说明

在 `config.py` 中配置：
- 监控股票列表
- 提醒提前天数
- 关注的事件类型
