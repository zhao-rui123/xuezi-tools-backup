# 直流电缆选型计算技能包

## 功能
储能/光伏直流系统电缆选型计算

## 使用方法
```
计算直流电缆选型
- 功率: X kW
- 电压: X V (最低/标准)
- 电缆规格: X mm²
- 环境温度: X ℃
- 敷设方式: 空气中/电缆沟
- 并线根数: X根
```

## 计算标准
基于GB/T 16895.15-2002 / IEC 60364-5-523标准

## 参数表
### 1. 1~3kV XLPE电缆空气中载流量(铜芯)
| 截面(mm²) | 载流量(A) |
|-----------|-----------|
| 70        | 244       |
| 95        | 296       |
| 120       | 340       |
| 150       | 388       |
| 185       | 436       |
| 240       | 643       |
| 300       | 736       |

### 2. 环境温度校正系数(90℃电缆)
| 温度(℃) | 系数 |
|---------|------|
| 30      | 1.00 |
| 35      | 0.96 |
| 40      | 0.92 |
| 45      | 0.94 |
| 50      | 0.89 |

### 3. 并线敷设校正系数(s=d)
| 根数 | 系数 |
|------|------|
| 2    | 0.90 |
| 3    | 0.85 |
| 4    | 0.82 |
| 5    | 0.80 |
| 6    | 0.79 |

### 4. 敷设方式校正
| 方式 | 系数 |
|------|------|
| 空气中 | 1.00 |
| 电缆沟(通风好) | 0.90 |
| 电缆沟(通风差) | 0.85 |

## 计算公式
```
I_load = P / U_min
I_cable = I_base × K_temp × K_install × K_parallel × N
```

## 示例
功率1562kW, 电压1164.8V, 240mm², 45℃, 电缆沟, 3根并线:
- 负载电流: 1341A
- 单根校正载流: 643 × 0.94 × 0.90 = 544A
- 总载流量: 544 × 3 × 0.85 = 1387A ✅
