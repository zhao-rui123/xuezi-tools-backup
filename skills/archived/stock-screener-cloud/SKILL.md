# Stock Screener Cloud - 云端股票筛选

基于MACD和KDJ技术指标的金叉选股工具，数据和缓存放在云服务器。

## 功能特性

### 📊 选股指标
| 指标 | 说明 |
|------|------|
| **MACD金叉** | DIF从下往上穿过DEA |
| **KDJ金叉** | K线从下往上穿过D线 |

### ⭐ 股票等级
| 等级 | 条件 | 信号强度 |
|------|------|---------|
| **A级** | MACD + KDJ 双金叉 | ⭐⭐⭐⭐⭐ |
| **B级** | 仅MACD金叉 | ⭐⭐⭐⭐ |
| **C级** | 仅KDJ金叉 | ⭐⭐⭐ |

### 📅 筛选时间范围
- 近1个月
- 近2个月（默认）
- 近3个月
- 近6个月

## 使用方法

```
选股 [时间范围]
- 选股 1个月
- 选股 2个月
- 选股 3个月
- 选股 6个月

筛选A级股票
筛选B级股票
筛选C级股票

查看选股结果
```

## 技术架构

- **数据源**：akshare（A股实时行情）
- **缓存位置**：云服务器 /opt/stock-screener/
- **调用方式**：通过SSH远程执行Python脚本
