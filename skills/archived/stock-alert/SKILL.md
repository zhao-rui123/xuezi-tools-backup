# Stock Alert - 股票预警系统

## 功能概述

实时监控自选股异常信号，满足条件立即推送提醒。

## 预警类型

| 预警 | 触发条件 | 级别 |
|------|---------|------|
| 🔴 突破MA20 | 价格向上突破20日均线 | warning |
| 🔴 跌破MA20 | 价格向下跌破20日均线 | warning |
| 🟡 大涨提醒 | 单日涨幅超过5% | info |
| 🔴 大跌提醒 | 单日跌幅超过5% | warning |
| 🟢 近期新高 | 创20日最高价 | info |
| 🔴 近期新低 | 创20日最低价 | warning |

## 使用方法

### 手动检查

```bash
cd ~/.openclaw/workspace/skills/stock-alert
python3 alert_engine.py
```

### 集成到日报

在股票日报推送前自动运行预警检查。

### 定时监控

可以设置每30分钟检查一次：
```bash
# crontab 示例（交易时段每30分钟）
*/30 9-15 * * 1-5 ~/.openclaw/workspace/skills/stock-alert/alert_engine.py
```

## 配置说明

在 `alert_config.py` 中配置：
- 监控股票列表
- 预警阈值（如涨幅阈值、均线周期）
- 预警级别对应的通知方式

## 数据来源

- 新浪财经（实时行情）
- 腾讯财经（K线数据）

## 注意事项

- 预警基于实时数据，可能有延迟
- 建议结合其他分析工具使用
- 不包含任何个人持仓信息
