# 文件交接工作流程

## 概述
通过下载中心进行文件交接，解决飞书无法发送大文件或特定格式文件的问题。

## 工作流程

### 1. 上传文件
- 将生成的 Word/Excel/PDF 等文件上传到下载中心
- 路径：`/usr/share/nginx/html/downloads/`
- 或通过后台上架功能上传

### 2. 通知下载
- 提供下载链接给用户
- 格式：`http://106.54.25.161/downloads/文件名`
- 告知文件大小和简要说明

### 3. 用户下载
- 用户访问下载中心
- 找到对应文件点击下载
- 如需管理权限，使用管理员密码：zr041412150

### 4. 文件管理
- 用户下载后可选择删除或保留
- 重要文件可锁定防止误删除
- 定期清理不需要的文件

## 管理员功能

### 上传文件
1. 进入下载中心：http://106.54.25.161/downloads/
2. 点击"管理模式"
3. 输入密码：zr041412150
4. 点击"+ 上传文件"选择文件
5. 上传成功后通知用户

### 锁定文件
- 防止重要文件被误删除
- 点击"锁定"按钮
- 锁定后删除按钮变为不可用
- 需先解锁才能删除

### 删除文件
- 点击"删除"按钮
- 确认删除操作
- 文件从服务器永久删除

## 注意事项

1. **文件命名**：使用中文或英文，避免特殊字符
2. **文件大小**：建议不超过 100MB
3. **敏感文件**：及时删除或锁定，避免长期存放
4. **定期备份**：重要文件在下载中心做好备份
