# China Enterprise Tax - 中国企业税务计算器

## 功能概述

专门针对中国企业税务的计算工具：
- ✅ **增值税计算** - 销项/进项/应交增值税
- ✅ **企业所得税** - 支持一般企业、小微企业、高新技术企业
- ✅ **附加税费** - 城建税、教育费附加、地方教育附加
- ✅ **印花税** - 销售合同、资金账簿
- ✅ **税负分析** - 税负率计算与税务筹划建议

## 税率支持

### 企业所得税
| 企业类型 | 税率 | 说明 |
|---------|------|------|
| 一般企业 | 25% | 标准税率 |
| 小微企业 | 5% | 实际税负（100万以下减按25%×20%） |
| 高新技术企业 | 15% | 优惠税率 |
| 西部大开发 | 15% | 优惠税率 |

### 增值税
- 一般纳税人：13%、9%、6%
- 小规模纳税人：3%（目前优惠1%）

### 附加税费
| 税费 | 市区 | 县城 | 其他 |
|------|------|------|------|
| 城建税 | 7% | 5% | 1% |
| 教育费附加 | 3% | 3% | 3% |
| 地方教育附加 | 2% | 2% | 2% |

## 使用方法

### Python调用

```python
from enterprise_tax_calculator import ChinaEnterpriseTaxCalculator, format_tax_report

# 创建计算器（市区、小微企业）
calc = ChinaEnterpriseTaxCalculator(
    location='市区',           # 市区/县城/其他
    enterprise_type='小微企业'  # 一般企业/小微企业/高新技术企业
)

# 计算税费
result = calc.calculate_all(
    revenue=1000000,           # 营业收入（不含税）
    cost=600000,               # 营业成本（不含税）
    operating_expenses=150000,  # 期间费用
    vat_output_rate=0.13,      # 销项税率
    vat_input_rate=0.13,       # 进项税率
)

# 输出报告
print(format_tax_report(result, '市区', '小微企业'))
```

## 适用场景

- 🏢 **初创企业** - 了解税费成本，规划财务
- 📊 **年度预算** - 预估全年税费支出
- 💰 **税务筹划** - 对比不同企业类型的税负
- 📋 **投资决策** - 评估项目税后收益

## 注意事项

1. **简化计算**：印花税按简化模型计算，实际以合同金额为准
2. **优惠政策**：小微企业优惠为简化计算，实际以税务局核定为准
3. **特殊情况**：不含即征即退、免税等特殊政策
